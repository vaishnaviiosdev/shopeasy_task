# ShopEasy Task

ShopEasy is a simple iOS application.


## Task Details

- Make use of appropriate UI components from the iOS framework.
- Home screen have a gradient-themed toolbar.
- Marking a favorite must animate the icon.
- Fonts and colours  of custom choice.
- Loading Data from given JSON file and it is locally stored as data source, using Core Data
- Categories menu and listing both must support expand & collapse behavior.
- User should be able to mark or unmark their favorite items and it should reflect across
the app.
- User should be able to add or remove multiple items to the cart and it should reflect
across the app.
- The item count displayed on the cart icon in the toolbar should be updated on user
action.
- Using Core Data to persist the product data, favorite items & cart items in the offline mode.
- Use Swift programming language following MVC Design pattern.
- Handling the various errors and exceptions gracefully.
